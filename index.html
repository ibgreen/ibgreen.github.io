Open Source Projects, WebGL Visualization @ Uber

I've spent a big part of my last two years working with the amazing Uber Visualization Team on these libraries. If you are doing high-performance geospatial visualizations on the Web, and you like to use the latest Web technologies, they are definitely worth checking out.
  • The vis.gl Framework Suite
  • deck.gl - WebGL Framework for geospatial visualization of big data
  • luma.gl - WebGL2 Components for high performance rendering and GPGPU compute
  • math.gl - A JavaScript 3D math library that leverages ES6 Array inheritance

Open Source Projects, C++/Cocoa

These are older personal projects I made for fun, to experiment with new techniques and technologies. I haven't looked at them for a long time, they are now likely in various states of (dis)repair. Most recent development on these was done in Xcode on MacOS.
  • libcocoawrap - A "zero-overhead" C++ interface to the Objective-C libraries (UIKit etc)
  • libsslstream - A type and exception safe C++14 API wrapper for SSL's (Secure Sockets Layer) clunky legacy C API
  • libexecstream - A C++14 version of the execstream module
  • libnpapiwrap - A modern C++ wrapper for the now defunct NPAPI (native browser plugin API)
  • libmassive - A cross-platform C++ UI component library, originally for Windows, recently mainly focusing on iOS
